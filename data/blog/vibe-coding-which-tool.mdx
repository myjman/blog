## 매주 새로운 도구가 나온다

2026년 2월 기준, AI 코딩 도구가 8개 넘게 나와 있다. 매달 새로운 도구가 등장하고, 기존 도구는 매주 업데이트된다. Cursor가 2.0을 내놓고, GitHub Copilot이 에이전트 모드를 추가하고, OpenAI가 Codex를 독립 제품으로 런칭했다.

문제는 도구마다 작동 방식이 다르다는 것이다. 어떤 건 에디터 안에서 돌아가고, 어떤 건 터미널에서 돌아가고, 어떤 건 브라우저에서 돌아간다. 가격도 무료부터 월 $200까지 천차만별이다.

이 글은 정리다. 세 가지 유형으로 나누고, 유형별 대표 도구를 비교하고, 마지막에 "나한테 맞는 도구"를 골라본다.

---

## 유형 1: 에디터 안에서 — Cursor, Windsurf, Copilot

코드 에디터(VS Code 같은 프로그램) 안에서 AI가 돌아가는 방식이다. 코드를 치면 AI가 다음 줄을 제안하고, 채팅창에서 지시하면 파일을 직접 수정한다.

![GitHub Copilot — 코드를 치면 회색 텍스트로 다음 줄을 제안한다](/static/images/vibe-coding-101/copilot-suggest.png)

**GitHub Copilot**은 가장 먼저 나왔고, 사용자가 가장 많다. 2,000만 명이 넘는다. 무료 플랜도 있고, 유료는 월 $10으로 가장 저렴하다. 코드 자동완성 + 간단한 채팅이 주력이다. 최근 에이전트 모드를 추가해서 파일을 직접 수정하는 것도 가능해졌다.

![Cursor 2.0 — AI 에이전트가 코드를 계획하고 직접 수정한다](/static/images/vibe-coding-101/cursor-agent.png)

**Cursor**는 한 단계 더 나간다. AI가 제안만 하는 게 아니라 직접 여러 파일을 동시에 수정한다. "이 컴포넌트에 다크 모드 추가해줘"라고 말하면 관련 파일을 전부 찾아서 고친다. 2년 만에 연매출 **$5억**을 달성한 이유가 있다. 바이브 코딩 도구 중 가장 인기가 많다.

![Windsurf Cascade — AI가 편집 히스토리를 추적하며 코딩한다](/static/images/vibe-coding-101/windsurf-cascade.png)

**Windsurf**(구 Codeium)는 가격 대비 성능이 좋다. 무료 플랜에서도 25크레딧을 주고, 유료는 월 $15로 Cursor보다 싸다. Cascade라는 에이전트가 대화 맥락을 유지하면서 코드를 수정한다.

| 도구     | 무료          | 유료 가격 | 강점                    | 약점               |
| -------- | ------------- | --------- | ----------------------- | ------------------ |
| Copilot  | 2,000 완성/월 | $10/월    | 가장 싸고 안정적        | 에이전트 기능 약함 |
| Cursor   | 50 요청/월    | $20/월    | 에이전트 최강, 인기 1위 | 무료 플랜 제한적   |
| Windsurf | 25 크레딧/월  | $15/월    | 가성비 좋음             | 생태계 작음        |

## 유형 2: 터미널에서 — Claude Code, Codex

에디터가 필요 없다. 터미널(검은 화면)에서 대화하듯이 코딩한다. "블로그 만들어줘"라고 말하면 파일 생성, 코드 작성, 빌드, 테스트, 배포까지 전부 한다.

![Claude Code — 터미널에서 자연어로 코딩한다](/static/images/vibe-coding-101/claude-code-og.jpg)

**Claude Code**는 Anthropic의 터미널 기반 AI 코딩 도구다. 이 블로그(오늘의 바이브)도 Claude Code로 만들었다. 터미널에서 대화하면서 디자인을 잡고, 글을 쓰고, 배포했다. 자유도가 가장 높지만, 터미널에 익숙해야 한다는 진입장벽이 있다. Claude Pro($20/월) 이상 구독이 필요하다.

![OpenAI Codex CLI — 터미널에서 GPT-5.3으로 코딩한다](/static/images/vibe-coding-101/codex-cli.png)

**Codex**는 OpenAI의 대응마. Claude Code와 비슷하게 터미널에서 동작한다. GPT-5.3을 쓰고, ChatGPT Plus($20/월) 이상이 필요하다. 샌드박스 환경에서 안전하게 코드를 실행하는 게 특징이다.

| 도구        | 무료   | 유료 가격     | 강점                           | 약점                       |
| ----------- | ------ | ------------- | ------------------------------ | -------------------------- |
| Claude Code | 없음   | $20/월 (Pro)  | 자유도 최고, 에이전트 성능 1위 | 무료 없음, 터미널 필요     |
| Codex       | 제한적 | $20/월 (Plus) | 샌드박스 안전, GPT-5.3         | 터미널 필요, 생태계 새로움 |

터미널형의 핵심은 **자유도**다. 에디터형이 "코드를 대신 짜주는 도우미"라면, 터미널형은 "프로젝트를 통째로 맡길 수 있는 에이전트"에 가깝다. 대신 터미널을 한 번도 열어본 적 없다면 진입장벽이 높다.

## 유형 3: 브라우저에서 — Lovable, Bolt, Replit

코드를 아예 안 보고 싶다면 이 도구들이다. 웹 브라우저에서 "할일 관리 앱 만들어줘"라고 입력하면 완성된 앱이 나온다. 설치할 것도 없다.

**Lovable**은 비개발자 시장의 대표 주자다. 출시 8개월 만에 연매출 $1억을 달성했고, 사용자의 **63%가 비개발자**다. 디자인을 Figma에서 스크린샷 찍어 붙여넣으면 그대로 만들어준다. 무료 플랜은 일 5크레딧(월 30), 유료는 $25/월.

**Bolt.new**은 StackBlitz 팀이 만든 브라우저 기반 빌더다. 무료 플랜이 넉넉하다(일 15만 토큰). AI가 코드를 생성하면 브라우저에서 바로 실행되는 걸 볼 수 있다. 유료는 $20/월.

**Replit**은 원래 온라인 코딩 환경이었는데, Agent 3를 붙이면서 AI 빌더로 변신했다. "앱 만들어줘"라고 말하면 에이전트가 코드를 짜고, 같은 화면에서 바로 실행된다. 무료는 에이전트 체험판 수준이고, 유료는 $20/월.

| 도구     | 무료         | 유료 가격 | 강점                         | 약점                   |
| -------- | ------------ | --------- | ---------------------------- | ---------------------- |
| Lovable  | 월 30크레딧  | $25/월    | 비개발자 최적화, 디자인→코드 | 복잡한 앱은 한계       |
| Bolt.new | 일 15만 토큰 | $20/월    | 무료 넉넉, 즉시 실행         | 대규모 프로젝트 어려움 |
| Replit   | 체험판 수준  | $20/월    | 올인원(코딩+실행+배포)       | 무료 플랜 제한적       |

## 전체 가격 비교

한눈에 보는 가격표다. 무료 플랜이 있는지, 유료는 얼마인지.

| 도구            | 유형     | 무료   | 유료 시작가 | 비고              |
| --------------- | -------- | ------ | ----------- | ----------------- |
| **Copilot**     | 에디터   | O      | $10/월      | 가장 저렴         |
| **Windsurf**    | 에디터   | O      | $15/월      | 가성비            |
| **Cursor**      | 에디터   | O      | $20/월      | 인기 1위          |
| **Claude Code** | 터미널   | X      | $20/월      | 에이전트 성능 1위 |
| **Codex**       | 터미널   | 제한적 | $20/월      | GPT-5.3           |
| **Bolt.new**    | 브라우저 | O      | $20/월      | 무료 넉넉         |
| **Replit**      | 브라우저 | O      | $20/월      | 올인원            |
| **Lovable**     | 브라우저 | O      | $25/월      | 비개발자 63%      |

대부분 **$20/월** 선에 모여 있다. 예외는 Copilot($10)과 Windsurf($15)가 싸고, Lovable($25)이 비싸다. 무료로 시작할 수 있는 도구가 대부분이니, 돈 쓰기 전에 먼저 써보는 걸 추천한다.

## 나한테 맞는 도구 고르기

"어떤 유형의 사람인가"에 따라 답이 달라진다.

**코딩을 전혀 모른다 → Lovable 또는 Bolt.new**
브라우저에서 말로 시키면 된다. 설치할 것도 없고, 코드를 볼 필요도 없다. Lovable이 디자인 퀄리티가 높고, Bolt.new은 무료 플랜이 넉넉하다.

**코딩을 배우면서 하고 싶다 → Cursor 또는 Replit**
코드가 어떻게 생겼는지 보면서 AI한테 시키고 싶다면 Cursor. 설치 없이 브라우저에서 하고 싶다면 Replit. 둘 다 AI가 코드를 짜주지만, 코드를 읽을 수 있는 화면이 항상 보인다.

**이미 개발자다 → Claude Code 또는 Cursor**
터미널에 익숙하고 자유도를 원하면 Claude Code. IDE 안에서 편하게 쓰고 싶으면 Cursor. 둘 다 에이전트 모드가 강력하다.

**돈을 쓰기 싫다 → Copilot Free 또는 Bolt.new Free**
Copilot 무료는 월 2,000번 자동완성 + 50번 AI 요청. 가벼운 코딩에는 충분하다. 앱을 만들고 싶으면 Bolt.new 무료가 일 15만 토큰으로 넉넉하다.

**한 줄 요약:**

| 나는...                          | 추천 도구        |
| -------------------------------- | ---------------- |
| 코딩 모름, 앱 만들고 싶음        | **Lovable**      |
| 코딩 배우면서 하고 싶음          | **Cursor**       |
| 개발자, 자유도 원함              | **Claude Code**  |
| 돈 쓰기 싫음                     | **Copilot Free** |
| 일단 빨리 뭐라도 만들어보고 싶음 | **Bolt.new**     |

## 도구보다 중요한 건 시작하는 것이다

도구 비교에 시간을 너무 쓰지 않아도 된다. 어차피 한 도구에 정착하면 나중에 바꾸기 쉽다. Cursor에서 시작했다가 Claude Code로 넘어가도 되고, Lovable로 빠르게 프로토타입 만들고 나중에 Cursor로 가져와서 다듬어도 된다.

대부분 무료로 시작할 수 있다. 일단 하나 골라서 써보자. 이 시리즈 다음 글에서는 Claude Code를 설치하고 첫 코드를 짜는 과정을 단계별로 따라해본다.
