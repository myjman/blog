## 에러 코드 하나가 바꾼 분위기

2026년 2월 첫째 주말, Vertex AI 로그에서 이상한 게 발견됐다. 존재하지 않는 모델 ID를 쿼리하면 404가 돌아온다. 당연한 거다. 그런데 `claude-sonnet-5@20260203`을 쿼리하니 **403**이 돌아왔다.

404는 "그런 거 없다"는 뜻이다. 403은 "있긴 한데 너한테는 안 보여준다"는 뜻이다.

이 차이 하나가 커뮤니티를 뒤집었다.

![Sonnet 5 유출 정보 요약](/static/images/sonnet5/sonnet5-og.png)

## 발견한 사람: Ben Taleb Jr.

최초 발견자는 **@macintoch**이라는 핸들의 Ben Taleb Jr.다. 그는 Google Cloud의 **Model Garden 엔드포인트**를 모니터링하는 스크립트를 돌리고 있었다.

방법은 단순했다:

- **대조군:** 가짜 모델 ID `claude-sonnet-99` → **404 Not Found** (존재하지 않음)
- **실험군:** 유출된 모델 ID `claude-sonnet-5` → **403 Forbidden** (존재하지만 접근 거부)

대조 실험의 교과서적인 설계다. 404와 403의 차이는 서버가 해당 리소스를 인식하고 있느냐 아니냐의 차이이므로, 403이 나왔다는 건 Google 인프라에 이미 해당 모델이 스테이징되어 있다는 뜻이 된다.

## 코드네임 Fennec

유출된 내부 코드네임은 **Fennec**이다. 페넥여우는 몸에 비해 귀가 비정상적으로 크다. 커뮤니티에서는 이를 루머 스펙의 **100만 토큰 컨텍스트 윈도우**에 빗대고 있다. Opus 4.5의 200K 대비 5배. 큰 귀 = 큰 입력 용량이라는 해석이다.

참고로 구글의 Gemini 3.5 내부 코드네임은 "Snow Bunny"다. Fennec이 Snow Bunny보다 한 세대 앞선 모델이라는 주장도 있다.

![AI 모델 경쟁 일러스트](/static/images/sonnet5/vertex-ai-log.jpg)

## 루머 스펙

여러 독립적인 소스에서 반복되는 수치들이 있다. 아직 Anthropic의 공식 확인은 없다.

| 항목 | 루머 스펙 |
|------|----------|
| 모델 ID | `claude-sonnet-5@20260203` |
| SWE-Bench | **82.1%** |
| 컨텍스트 윈도우 | **100만 토큰** (Opus 4.5의 5배) |
| 가격 | Opus 4.5 대비 **약 50% 저렴** |
| 성능 수준 | Opus 4.5급 |
| 특수 기능 | "Dev Team" 모드 (추정), 구조화된 비주얼 생성 |

한 개발자가 4,000줄짜리 Rust 리팩토링을 Sonnet 5에 던졌더니, GPT-5.2와 달리 전체 의존성 트리를 할루시네이션 없이 추적했다는 보고도 있다.

## Opus 4.5와 비교

|  | Opus 4.5 (현재) | Sonnet 5 Fennec (루머) |
|---|---|---|
| 컨텍스트 | 200K 토큰 | 100만 토큰 |
| SWE-Bench | - | 82.1% |
| 가격 | 기준점 | ~50% 저렴 |
| 포지셔닝 | 플래그십 | 플래그십급 성능, 중간 가격대 |
| 출시 | 2025년 11월 | 미출시 |

Opus 4.5가 나온 지 겨우 10주밖에 안 됐다. 이 짧은 간격이 오히려 의심을 사는 부분이다. 제로 마케팅으로 플래그십을 깜짝 출시하는 건 Anthropic 스타일이 아니다.

## 바이브 코더에게 의미

현실적인 질문: 지금 쓰는 모델 스택을 바꿔야 하나?

**아직 아니다.**

- 공식 발표가 없다
- 커뮤니티 신뢰도 평가는 60-70% "아마 진짜"
- 최초 근거는 트위터 스크린샷이었고, Vertex AI 로그 원본에 대한 직접 접근 증거는 없다
- Gemini 3.5 Snow Bunny, GPT-5.2 업그레이드 등 경쟁사 루머도 동시에 돌고 있다. 리크가 시장 압력을 만드는 시기다

다만 100만 토큰 컨텍스트가 사실이라면 게임이 바뀐다. 지금까지 "코드베이스를 쪼개서 넣어야 하는" 제약이 사라지는 거다. Anthropic 블로그는 주시할 가치가 있다.

## Anthropic은 아직 아무 말 없다

Anthropic은 이 루머에 대해 일체 반응하지 않았다. 확인도, 부인도 없다.

확실한 건 하나다: Google Vertex AI 인프라 어딘가에 `claude-sonnet-5`라는 이름의 리소스가 존재하고, 그것은 404가 아니라 403을 반환한다.

나머지는 기다려 봐야 안다.
